# ``BbAppNewPost``

New post creation functionality and user interface.

## Overview

The BbAppNewPost library provides comprehensive post creation capabilities. It includes the new post view interface with rich text editing support, intent sheet presentation for modal post creation workflows, the service layer that handles post submission and validation, and state management for the post creation process. This module enables users to create new discussion topics and content.
